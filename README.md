# Portfolio Project - Full Stack App

A high-performance portfolio with a robust Admin Dashboard, built for showcasing your work, services, and handling client leads/bookings.

## 🚀 Features

### Admin Dashboard
- **Overview Area**: Track visitor stats (real analytics implementation planned), leads, and booked sessions.
- **Hero & About Management**: Easily edit your landing page content, roles, and profiles.
- **Projects & Services**: Add, edit, or remove showcase items and pricing effortlessly.
- **Client Testimonials** (NEW!): Manage customer feedback with star ratings.
- **Direct Image Uploads** (NEW!): Upload images directly from your device to the server. No more copy-paste URLs!

### Core Sections
- **Interactive Hero**: Dynamic roles and premium animations.
- **Project Showroom**: Filterable project gallery with detailed modal views.
- **Testimonial Slider**: Premium feedback showcase.
- **Consultation Booking**: Instant session scheduling.
- **Contact Form**: Direct email lead capture.

## 🛠️ Technology Stack
- **Frontend**: React (Vite), TypeScript, Tailwind (Glassmorphism design), Motion (framer-motion).
- **Backend**: Node.js (Express), MongoDB (Mongoose), Multer (Image uploads).
- **Email**: Nodemailer (GMAIL SMTP).

## 💻 Local Development

### 1. Backend Setup
1. `cd backend`
2. `npm install`
3. Create `.env` from `.env.example`
4. `npm run dev`

### 2. Frontend Setup
1. `cd frontend`
2. `npm install`
3. Create `.env` from `.env.example`
4. `npm run dev`
